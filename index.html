Project Overview

GOR Garuda merupakan salah satu penyedia layanan penyewaan lapangan bulu tangkis (badminton) di Kota Mataram. Lapangan di GOR ini menggunakan standar garis dan ukuran sesuai peraturan bermain bulu tangkis, dan lapangan dibangun dengan kualitas terbaik sehingga pemain dapat berolahraga dengan nyaman. Project web ini saya buat untuk memudahkan proses penyewaan lapangan di GOR Garuda, sekaligus untuk memenuhi tugas akhir mata kuliah Pemrograman Web di kampus. Web ini saya buat menggunakan Laravel 9 dan framework Bootstrap. Dibawah saya lampirkan beberapa tampilan dari webnya.
